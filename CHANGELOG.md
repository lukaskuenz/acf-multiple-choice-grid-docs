# Changelog

## 1.0.0 – Initial release

- Initial public release of **ACF Multiple Choice Grid**
- Added custom ACF field type for multiple choice grids
- Radio-style matrix with exactly one selection per row
- Rows and columns configurable using `value : label` syntax
- Supports ACF-style return formats:
  - Value
  - Label
  - Both (Array)
- Supports multiple return shapes:
  - Columns only
  - Row → Column map
  - Pair strings (e.g. `row:column`)
- Compatible with `acf_form()` and frontend submissions
- Semantic, accessible table markup
- No frontend styling assumptions
